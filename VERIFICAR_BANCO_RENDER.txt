========================================
✅ COMO VERIFICAR O NOME DO BANCO NO RENDER
========================================

Siga estes passos para encontrar o nome correto:

========================================
PASSO 1: IR PARA O BANCO
========================================

1. No Dashboard do Render
2. Clique no seu banco PostgreSQL "cshub-db"
3. Você verá a página de detalhes do banco

========================================
PASSO 2: PROCURAR O NOME DO BANCO
========================================

Na página do banco, procure por uma destas seções:

OPÇÃO A: "Connection Info" ou "Connection Details"
- Mostra: Host, Port, Database, User, Password
- O nome do banco está em "Database"

OPÇÃO B: "Internal Database URL"
- Formato: postgresql://user:pass@host:port/database
- O nome do banco está DEPOIS da última barra (/)
- Exemplo: postgresql://.../cshub_db
  → O nome é: cshub_db

OPÇÃO C: "Database" (campo simples)
- Mostra diretamente o nome do banco

========================================
PASSO 3: COMPARAR COM VARIÁVEIS
========================================

1. Vá para seu Web Service "cshub"
2. Clique em "Environment"
3. Encontre a variável "DB_NAME"
4. Compare com o nome que você encontrou no banco

Se forem DIFERENTES:
→ Edite DB_NAME para ter o nome EXATO do banco

========================================
EXEMPLO:
========================================

Se no banco você vê:
- Database: cshub_db

Então na variável DB_NAME deve ser:
- DB_NAME = cshub_db

(NÃO "cshub", mas "cshub_db" com underscore!)

========================================
DICA:
========================================

O Render às vezes adiciona sufixos ao nome do banco.
Verifique sempre o nome REAL no banco, não o que
você digitou ao criar!
