O QUE FAZER AGORA:
========================================

OPÇÃO 1: USAR BUILD COMMAND (MAIS FÁCIL)
-----------------------------------------

1. No Render, vá em "Settings" do seu Web Service

2. Encontre o campo "Build Command"

3. Altere para exatamente isto:

   pip install -r requirements.txt && python manage.py collectstatic --noinput && python manage.py init_production

4. Clique em "Save Changes"

5. O Render fará um novo deploy

6. Aguarde o deploy terminar (5-10 minutos)

7. Pronto! As migrações e o usuário serão criados automaticamente!

8. Depois que funcionar, você pode voltar o Build Command para:
   pip install -r requirements.txt && python manage.py collectstatic --noinput

========================================
OPÇÃO 2: USAR VARIÁVEL DE AMBIENTE
-----------------------------------

1. No Render, vá em "Environment"

2. Adicione uma nova variável:
   Key: INIT_DB
   Value: true

3. Salve

4. Faça um novo deploy (Manual Deploy → Deploy latest commit)

5. Na primeira vez, as migrações serão executadas automaticamente

6. Depois, remova a variável INIT_DB ou mude para false
